GIT 操作

August 09, 2019 4562

git流程图

  • Workspace 工作区
  • Staging/Index：暂存区
  • Local Repository：本地仓库（可修改）
  • /refs/remotes：远程仓库的引用（不可修改）
  • Remote：远程仓库

  • git diff
  • git show
  • git rebase
  • git restore
  • git revert

git 特殊情况操作

  • 代码没完成切换分支
  • 本地临时代码不想提交，一次性清空
  • 已经提交的代码，不需要了
  • git reset –hard 命令，执行错误
